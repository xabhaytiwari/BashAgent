# BashAgent

An AI-powered DevOps assistant embedded directly in your Git Bash.

**Features:**
* **Brain:** Powered by Google Gemini & LangGraph.
* **Hands:** Can execute shell commands, write scripts, and manage files.
* **Dual Modes:**
    * **CLI:** `ai "list all python files"` (Quick tasks)
    * **Interactive:** `ai` (Full conversation with memory)
* **Safety:** Human-in-the-Loop approval required before executing any command.
